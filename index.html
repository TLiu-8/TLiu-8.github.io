Prom Ticketing System

Feb 2020

Multi-component software that allows students to buy tickets for Prom using a graphical interface. The algorithm assigns them to tables based on their friend preference and displays a graphical layout of the floorplan including all the tables and who is seated at each table.

ProMesh

July 2019

Environmentally friendly produce bags made from recycled PET plastic with smart technology (RFID chips that make inventory and checking out to be more efficient). Bags can be returned by customers the next time they go shopping, to be re-used. (similar to beer bottle system) Project created at SHAD Carleton by the Green House.

Marching Orders

Jan 2019

Single Player Challenge game where you give orders to each marching robot, and guide it to the matching-color house. If a robot falls, you will lose a heart. You don't lose a heart if a dog falls. Object-orientated program written in Java. ICS3U6 Final Project.

Ecosystem Simulator

Dec 2018

Object Oriented Program simulating the population dynamics of an ecosystem containing plant, sheep, and wolves in Java. Animals can mate with each other, grow over time, fight each other, eat each other, and the simulation ends when only one species is remaining.

Reflex Arc Model

Dec 2018

Electronic model of the reflex arc, stimulated by heat to cause the wooden leg to move. It consists of: a wooden box holding an Arduino and breadboard, a heat sensor sticking out of the box, a LCD panel located on the top surface of the box, a wooden platform with a motor controlling a “leg” that is connected to the Arduino, and a 3D printed spine with a LED at the top vertebrae resting on top of the box.
